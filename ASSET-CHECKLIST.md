# Memorial Weekend Website - Asset Checklist

## Required Images for the Website

Before running the website, download the best photos from your Facebook page and save them in the `images/` folder with these exact filenames:

### 1. **hero-bg.jpg**

- **Description**: Main background for the hero section (full viewport height)
- **Recommended Size**: 1920x1080px or larger
- **Find**: The "Field of Crosses" wide shot in landscape orientation
- **Requirements**: High resolution, dramatic lighting preferred

### 2. **mission-bg.jpg** (Optional)

- **Description**: Background for the "Words From a Warrior" mission section
- **Recommended Size**: 1920x1080px
- **Find**: Photo of a soldier/veteran saluting or a folded flag close-up
- **Requirements**: Should work with navy blue overlay

### 3. **schedule-run.jpg**

- **Description**: Thumbnail for the Saturday 5K Run event
- **Recommended Size**: 400x300px
- **Find**: Photo of the 5K Run start line or runners
- **Note**: Displays in schedule timeline

### 4. **schedule-carshow.jpg**

- **Description**: Thumbnail for the Saturday Car Show event
- **Recommended Size**: 400x300px
- **Find**: A cool classic car from a previous year's show
- **Note**: Displays in schedule timeline

### 5. **schedule-flyover.jpg**

- **Description**: Thumbnail for the Sunday Aerial Flyover
- **Recommended Size**: 400x300px
- **Find**: Helicopter or planes overhead during the flyover
- **Note**: Displays in schedule timeline

### 6-9. **gallery-1.jpg through gallery-4.jpg**

- **Description**: Gallery preview images (4 images total)
- **Recommended Size**: 800x600px each
- **Find**: The "Best of" photos showcasing:
  - `gallery-1.jpg`: Crowd shot or ceremony
  - `gallery-2.jpg`: Color guard or flag presentation
  - `gallery-3.jpg`: Emotional moment (candlelight vigil, wreath laying)
  - `gallery-4.jpg`: Community gathering or children's activities
- **Requirements**: Mix of formal ceremonies and community moments

### 10. **logo.png**

- **Description**: "Not Forgotten" or "American Legion" logo
- **Recommended Size**: 300x300px (square) or 500x200px (horizontal)
- **Find**: Use the logo you mentioned having
- **Requirements**: Transparent background (PNG format) preferred
- **Note**: Used in navbar and footer

### 11-14. **sponsor-1.png through sponsor-4.png** (or more)

- **Description**: Sponsor company logos
- **Recommended Size**: 300x150px each
- **Find**: Crop from "Thank You Sponsors" flyers on Facebook
- **Requirements**: Transparent backgrounds preferred, PNG format
- **Note**: Add as many as you have sponsors

---

## Quick Setup Instructions

1. Create an `images/` folder in your `veteranspage/` directory if it doesn't exist
2. Download photos from Facebook
3. Rename them exactly as listed above
4. Place them in the `images/` folder
5. Open `index-redesign.html` in your browser

---

## Image Optimization Tips

- **Format**: Use JPG for photos, PNG for logos
- **Compression**: Compress images to reduce file size (use tools like TinyPNG.com)
- **Resolution**: Use 72 DPI for web images
- **File Size**: Keep images under 500KB each for faster loading
