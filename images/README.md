# Place your logo here

To add your "Not Forgotten" logo:

1. Name your logo file: `logo.png` (or logo.jpg)
2. Place it in this `images` folder
3. The website will automatically display it in the navigation bar

Recommended logo specifications:

- Format: PNG (with transparent background) or JPG
- Size: 200-400 pixels wide
- Aspect ratio: Maintain original proportions
- File size: Under 500KB for faster loading

You can also add other images here for the photo gallery:

- memorial-2.jpg
- memorial-3.jpg
- etc.
